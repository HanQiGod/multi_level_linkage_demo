# multi_level_linkage_demo

Flutter 多级联动选择器示例项目，对应文章《灵活强大的适配器模式，轻松构建任意层级联动选择器》。

这个仓库现在不是单页长列表，而是：

- 首页负责文章导读和示例导航
- 每个联动场景都有独立详情页
- 详情页只展示当前示例，方便截图和文章分节引用

## 文章配套说明

本项目用于配合讲解 `flutter_picker_plus` 的以下能力：

- 用 `PickerItem` 构建任意层级树形数据
- 用 `PickerDataAdapter` 解耦界面与数据源
- 用 `selecteds` 做默认回显
- 用 `changeToFirst` 处理父子列联动
- 用同一套适配器模式承接静态数据和动态生成数据

## 页面结构

### 首页导航

首页展示 4 个示例入口，每个入口都包含：

- 场景标题
- 默认路径预览
- 亮点标签
- 独立详情页按钮

### 详情页

每个示例进入单独页面后，会展示：

- 当前示例说明
- 当前默认选中路径
- 打开选择器按钮
- 恢复默认选中按钮
- 文章要点回顾

这种结构比单页长列表更适合仓库展示，也更适合文章逐节截图。

## 当前内置示例

项目目前包含 4 个可直接运行的联动示例：

1. 商品分类三级联动
2. 省 / 市 / 区地址联动
3. 生日年 / 月 / 日联动
4. 组织架构四级联动

### 1. 商品分类三级联动

用于对应文章里的基础示例，重点演示：

- `PickerItem` 的树形嵌套
- `PickerDataAdapter` 的接入方式
- `changeToFirst` 的联动效果
- `getSelectedValues()` 回填选中结果

### 2. 省 / 市 / 区地址联动

用于模拟真实表单中的地址选择，重点演示：

- 地址树如何映射为联动数据
- 选择结果如何回填到页面
- 同一套组件如何复用于另一种业务数据

### 3. 生日年 / 月 / 日联动

用于演示动态生成数据，重点演示：

- 运行时代码生成联动树
- 闰年 2 月的天数变化
- `columnFlex` 调整列宽
- `looping` 改善滚动体验

### 4. 组织架构四级联动

用于强调“任意层级”这个核心卖点，重点演示：

- 四级联动数据结构
- 默认回显到第四层节点
- 组织树 / 渠道树 / 品类树的典型写法
- 四级结构仍可复用同一个示例面板组件

## 依赖

核心依赖如下：

```yaml
dependencies:
  flutter_picker_plus: ^1.5.5
```

当前实际解析版本见 `pubspec.lock`。

## 运行方式

安装依赖：

```bash
flutter pub get
```

启动项目：

```bash
flutter run
```

运行验证：

```bash
flutter test
flutter analyze
```

## 代码结构

主要代码位于 `lib/src/`：

- `app.dart`：应用入口与主题配置
- `demo_models.dart`：数据模型定义
- `demo_data.dart`：示例数据、动态生日树和示例配置
- `demo_ui.dart`：首页导航、详情页和联动面板 UI

根目录 `lib/main.dart` 只保留运行入口，方便测试和引用。

## 适合怎样引用到文章里

如果你要继续写这篇系列文章，这个仓库现在比较适合以下引用方式：

1. 文章开头引用首页导航截图，说明仓库包含 4 个完整案例
2. 每一节正文引用对应详情页截图
3. 代码讲解时直接指向 `demo_data.dart` 和 `demo_ui.dart`
4. 需要强调“任意层级”时，直接展示四级组织架构页面
