# multi_level_linkage_demo

Flutter 多级联动选择器示例

## 页面结构

### 首页导航

首页展示 4 个示例入口，每个入口都包含：

- 场景标题
- 默认路径预览
- 亮点标签
- 独立详情页按钮

### 详情页

每个示例进入单独页面后，会展示：

- 当前示例说明
- 当前默认选中路径
- 打开选择器按钮
- 恢复默认选中按钮
- 文章要点回顾

这种结构比单页长列表更适合仓库展示，也更适合文章逐节截图。

## 当前内置示例

项目目前包含 4 个可直接运行的联动示例：

1. 商品分类三级联动
2. 省 / 市 / 区地址联动
3. 生日年 / 月 / 日联动
4. 组织架构四级联动

### 1. 商品分类三级联动

用于对应文章里的基础示例，重点演示：

- `PickerItem` 的树形嵌套
- `PickerDataAdapter` 的接入方式
- `changeToFirst` 的联动效果
- `getSelectedValues()` 回填选中结果

### 2. 省 / 市 / 区地址联动

用于模拟真实表单中的地址选择，重点演示：

- 地址树如何映射为联动数据
- 选择结果如何回填到页面
- 同一套组件如何复用于另一种业务数据

### 3. 生日年 / 月 / 日联动

用于演示动态生成数据，重点演示：

- 运行时代码生成联动树
- 闰年 2 月的天数变化
- `columnFlex` 调整列宽
- `looping` 改善滚动体验

### 4. 组织架构四级联动

用于强调“任意层级”这个核心卖点，重点演示：

- 四级联动数据结构
- 默认回显到第四层节点
- 组织树 / 渠道树 / 品类树的典型写法
- 四级结构仍可复用同一个示例面板组件
